# Prompt Documentation – CineStream Testing Suite

## 📌 Project Overview

This document contains the prompts used while enhancing the **CineStream – Movie Recommendation App** with a complete testing suite using Jest and React Testing Library.

The goal was to implement unit testing, interaction testing, and API mocking while maintaining clean and scalable code.

---

## 🧠 Prompt 1: Testing Setup

**Prompt:**
Add Jest and React Testing Library to my existing React project.
Provide installation commands and necessary configuration so that tests can run using `npm test`.

---

## 🧠 Prompt 2: Basic Unit Testing

**Prompt:**
Write test cases for React components like Button, MovieCard, and SearchBar.
Ensure the tests check if the components render correctly and display the right props/text.

---

## 🧠 Prompt 3: Interaction Testing

**Prompt:**
Write tests using React Testing Library to simulate user interactions such as typing in an input field and clicking a button.
Ensure the UI updates correctly after the interaction.

---

## 🧠 Prompt 4: API Testing with Mocking

**Prompt:**
Write tests for components that fetch movie data from an API.
Mock the API calls using Jest so that the tests do not make real network requests.
Test loading, success, and error states.

---

## 🧠 Prompt 5: Test Coverage

**Prompt:**
How can I generate a test coverage report in my React project using Jest?
Suggest ways to improve coverage to at least 70%.
